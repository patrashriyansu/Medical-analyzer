Figma Design Assets and Mockups
--------------------------------

This folder contains simple SVG mockups and a short guide to recreate or import the UI into Figma.

How to use:
- Open Figma and create a new file.
- Drag the SVG files from this folder into the canvas to use them as visual references.
- Recreate layouts in Figma using the colors and spacing in the SVGs.

Notes:
- These SVGs are lightweight mockups (auth and dashboard) to speed up visual design iterations.
- If you want a native Figma file, export your Figma project or share a Figma link and we can help integrate.
